# Identidade Visual

A identidade visual do HFinance usa ciano e dourado como sinais principais, com fundo claro, cards brancos e contraste adequado para uso diário.

## Paleta

- Ciano principal: `#00E5FF`.
- Dourado: `#FFD700`.
- Fundo principal: `#F4F7FB`.
- Cards: `#FFFFFF`.
- Texto principal: `#1F2937`.
- Texto secundário: `#6B7280`.
- Erro: `#EF4444`.
- Sucesso: `#22C55E`.
- Alerta: `#F59E0B`.

## Componentes

- DatePicker deve manter mês e ano legíveis em fundo claro.
- Formulários repetitivos devem preservar campos úteis e oferecer ação clara de limpar formulário.
- Tabelas não devem dominar a tela quando há poucos dados.
- Telas com conteúdo vertical devem usar área rolável.
- Botões destrutivos devem usar destaque de perigo e ficar visualmente separados das ações primárias.
- Notificações comuns devem aparecer como toasts compactos e empilhados no canto superior direito, com fechamento automático e manual.
- Confirmações devem permanecer integradas à janela, usar altura baseada no conteúdo e aceitar Enter para confirmar e Esc para cancelar.
- Gráficos financeiros devem ter título, legenda quando aplicável, eixos identificados, valores em reais, percentuais e tooltips.

## Ícone

O ícone usa fundo ciano, H maiúscula grande e detalhe dourado. Os arquivos esperados são:

- `src/main/resources/images/app-icon.svg`
- `src/main/resources/images/app-icon.png`
- `src/main/resources/images/app-icon.ico`
